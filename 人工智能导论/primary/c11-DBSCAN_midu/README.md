# ✨ 异常点检测：基于密度聚类（DBSCAN）的实验

## 📌 简介

本项目是一个教学型的异常点检测案例，使用密度聚类方法 DBSCAN 在二维数据中识别异常点。案例覆盖了数据生成、可视化、k-距离图选参、聚类结果展示、与真实标签对比、参数敏感性分析以及拓展思考。

## 🧠 为什么使用密度聚类做异常点检测

- 异常定义自然：低密度区域样本可直接视为噪声点。
- 无需预设簇数：DBSCAN 能自动发现簇结构。
- 适配任意形状簇：对非球形簇也能较好识别。

## 🧭 内容结构

- 环境准备与库导入（含中文显示与警告抑制设置）
- 生成示例数据（含异常点）
- 原始数据可视化
- k-距离图估计 eps
- 运行 DBSCAN 并识别异常点
- 聚类结果与异常点可视化
- 与真实标签对比（教学用）
- eps 参数敏感性观察
- 小结与拓展思考（含解题思路）

## ▶️ 如何使用

1. **准备环境**：建议使用 Python 3.9+。
2. **安装依赖**（在项目根目录）：
   ```bash
   pip install -r requirements.txt
   ```
3. **打开笔记本**：在 VS Code 或 Jupyter 中打开 [yichangjiance.ipynb](yichangjiance.ipynb)。
4. **顺序执行单元**：从上到下依次运行，逐步得到数据、图表与中间结果。
5. **可选练习**：修改 `eps`、`min_samples` 或异常点数量，观察结果变化。

> 提示：若图表中文显示异常，请确保系统安装了常见中文字体（如微软雅黑/黑体）。
