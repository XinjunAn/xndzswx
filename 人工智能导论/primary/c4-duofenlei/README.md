# 水果分类决策树教学案例

## ✨ 项目简介

本项目是一个多分类教学案例，使用模拟生成的水果数据集，通过决策树完成水果识别，并进行模型评估与可视化分析。内容覆盖从数据生成、训练、评估到可解释性分析的完整流程。

## 🧭 目录结构

```
c4-duofenlei/
├─ duofenlei-shuiguo.ipynb    教学主文档与完整实验流程
├─ data/
│  └─ fruit_dataset.csv       实验生成的数据集
├─ requirements.txt           依赖列表
└─ 案例背景书.docx
```

## 🧺 数据集说明

数据集为模拟生成的水果多分类数据，共 500 条样本、5 个类别（苹果、香蕉、橙子、柠檬、芒果）。
特征字段：mass（质量）、width（宽度）、height（高度）、color_score（颜色评分），标签字段为 fruit_name。

## 🛠️ 环境依赖

依赖在 requirements.txt 中，包含 numpy、pandas、matplotlib、seaborn、scikit-learn 以及 Jupyter 相关组件。

## 🚀 快速开始

1. 安装 requirements.txt 中的依赖。
2. 打开 duofenlei-shuiguo.ipynb。
3. 依次运行各单元，观察输出与图表。

## 🧪 实验流程

1. 导入库并完成基础设置（随机种子、中文字体）。
2. 生成模拟水果数据并保存到 data/fruit_dataset.csv。
3. 划分训练集与测试集。
4. 训练决策树（设置 max_depth 与 random_state）。
5. 预测与评估（准确率、分类报告）。
6. 可视化决策树结构。
7. 可视化数据分布（宽度 vs 高度）。
8. 混淆矩阵可视化。
9. 特征重要性分析。
10. 关键水果识别规则总结。
11. 拓展与思考（含解答思路）。

## 📊 结果产出

- 决策树结构图
- 混淆矩阵热力图
- 特征重要性可视化
- 规则总结与学习拓展

## 📝 备注

本案例使用模拟数据，若替换为真实采集数据，建议先做分布对比与必要的特征清洗。
